    * Create /api/contracts endpoints (list, filter, search)
    * Create /api/contract-chats endpoints
    * Create /api/prompts endpoints (user saved & system prompts)
    * Update server/routes.ts and storage.ts
* Sidebar Restructure with Hamburger Menu
    * Add hamburger menu icon (three lines → X animation) at top of sidebar
    * Create smooth sidebar collapse/expand animation
    * Maintain sidebar state in localStorage
    * Ensure logo remains visible in collapsed state
* Contract Chat History Search
    * Place search box below hamburger menu, above "Recent Contracts"

Key UI/UX Best Practices
* Progressive Disclosure: Prompts hidden until needed
* Familiar Patterns: ChatGPT-style interface users already know
* Smooth Animations: 300ms transitions for natural feel
* Clear Visual Hierarchy: Important actions prominently displayed
* Responsive Design: Adapts elegantly to all screen sizes
* * Accessibility: Keyboard navigation and ARIA labels
